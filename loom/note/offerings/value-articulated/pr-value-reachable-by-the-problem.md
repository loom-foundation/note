kind: pain-reliever
status: current

The value is stated against the need it answers and the commitments that meet it,
so a customer who arrives with a specific problem reaches the relevant value by that problem rather than by decoding a feature list,
and the answer they would otherwise raise as a support ticket is already where they looked for it.
