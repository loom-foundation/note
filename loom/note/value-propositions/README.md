# Value propositions

The authored pitches of Note's offerings: the customer-facing claims that product, marketing, and documentation copy restate from one source.

Each value proposition expresses one Offering, and the evidence behind it is the computed fit: the `relieves` and `creates` links from that Offering's pain relievers and gain creators to the pains and gains of the needs it serves.
A claim these links cannot back is a pitch writing cheques the fit cannot cash, and is reworded or cut.
