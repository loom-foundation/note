The on-ramp is a file, not a migration.

## The case

You should not have to stop building to start capturing intent.
Note adopts onto the system you already have, mid-flight.
The first artifact pays for itself the day you write it; recovered knowledge is recorded at honest confidence instead of waiting to be perfect.
Rigor rises with your stakes, and nothing you wrote before needs restructuring.
