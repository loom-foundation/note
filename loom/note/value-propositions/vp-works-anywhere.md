Works on any machine, agent, or pipeline, with nothing to install.

## The case

The intent is plain files, read and written with the editor, shell, and file access every machine already has.
A new engineer on a laptop, an AI agent boxed in a sandbox or a CI runner, and the pipeline itself all take part at once, with no tool to locate and set up first.
Nobody waits on an install, and nobody is left out because their environment cannot run it.
The same step that works on your machine works everywhere it lands.
