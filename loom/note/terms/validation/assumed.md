An unvalidated assumption or hypothesis: believed but not yet tested or confirmed.

Where an imagined problem or an untested value proposition starts.

## Inclusion test

If challenged, would the honest answer be "we believe this but have not validated it"?
If so, the claim is Assumed; if it rests on a reasoning chain it is Inferred, and if it was directly confirmed it is Observed.

## Relations

- *is a value of* [Validation](validation.md){id=note:term:13esbh9}

## Origin

General English ("assume": to take as true without proof, pending validation).

In this corpus's validation tiering, Assumed is the starting tier, the lowest of the five, marking a claim held explicitly until it is tested.
